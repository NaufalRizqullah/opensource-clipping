About Automation Clippers

OpenSource Clipping is a content automation suite designed to transform raw, long-form footage into highly engaging vertical clips (9:16) featuring word-by-word karaoke subtitles, face-tracking auto-framing, smart segment trimming, and dynamic stock B-rolls.

Automation Clippers (registered as Automation_Clppers in the Meta developer dashboard) is the specialized upload utility integrated into the OpenSource Clipping codebase. It securely connects to the Meta Graph API on behalf of authorized administrators to manage and publish generated short-form clips directly to Facebook Pages.

By connecting your Facebook Page credentials, the automation pipeline can:

  • Upload high-retention highlight reels to target pages.
  • Configure smart scheduling options to post the first clip immediately and queue subsequent reels at set intervals.
  • Set video metadata, including custom generated titles, descriptions, and hashtags.
  • Verify upload statuses and review logs to ensure API compliance.

Because this integration serves as a secure bridge for the CLI automation tool, public registrations are disabled. Use of this application is restricted to authorized administrators managing their respective Facebook Pages and assets.
